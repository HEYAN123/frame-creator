## 指令说明

```bash
$ frame-create init template-name project-name  根据已有模板初始化一个项目
$ frame-create config set <key> <value>  修改配置信息
$ frame-create config get [<key>]  查看配置信息
$ frame-create -v  查看当前版本信息
$ frame -h  查看帮助信息
```
